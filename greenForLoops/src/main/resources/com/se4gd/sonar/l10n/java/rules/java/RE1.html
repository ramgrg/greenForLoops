Do not concatenate string inside loops. If you have to, use StringBuilder instead.

Noncompliant Code Example

public String concatenateString(String[] sampleStrings){
    String finalString = "";
    for(String eachString: sampleStrings){
        finalString += eachString;
    }
    return finalString;
}
public String concatenateStringAlt(){
    String finalString = "";
    for(int i = 0; i < 50; i++){
        finalString += "sample string";
    }
    return finalString;
}

Compliant Solution

public String concatenateString(String[] sampleStrings){
    StringBuilder finalString = new StringBuilder();  
    for(String eachString: sampleStrings){
        finalString.append(eachString);
    }
    return finalString.toString();
}
